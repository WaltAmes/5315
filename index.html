What is Docker

Docker is used to make creating and running applications easier by using containers. The container packages up all of the required parts ensuring the application will run on other machines despite having different settings. Docker is similar to a VM but does not include a virtual operating system, instead the application relies on the host system but is shipped with libraries and dependencies which may not be running on the host computer. This method of packaging offers better performance than a VM. Source: https://opensource.com/resources/what-docker

What is Nginx

NGINX is a web server known for high performance and stability. Originally NGINX supported HTML but it has expanded to support more complex content. Given the level of performance NGINX has achieved it is also used as a reverse proxy and load balancer for slower systems such as databases and microservices. Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A Virtual Machine (VM) is a software based representation of an actual computer system. A VM runs in a window much like other applications giving users the functionality of the host computer. Source: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/ Source: https://www.vmware.com/solutions/virtualization.html

Who is Cloud Computing Service Provider - Digitalocean

Digital Ocean is a vendor that offers Infrastructure as a Service (IaaS) for cloud computing. Digital Ocean provides its users with private virtual machines and is heavily used by open source developers. Source: https://searchcloudcomputing.techtarget.com/definition/DigitalOcean
